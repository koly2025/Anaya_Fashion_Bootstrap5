
Anaya Fashion - Responsive HTML E-commerce Template
----------------------------------------------------

Thank you for purchasing/download Anaya Fashion Template.
This is a modern and responsive HTML e-commerce template suitable for fashion, beauty, and lifestyle stores.

STRUCTURE:
- index.html             -> Homepage
- about.html             -> About page
- contact.html           -> Contact form (HTML only)
- product1.html, etc.    -> Product detail pages
- man's.html,   etc      -> Multiple product pages
- cart.html,checkout.html,Heart.html   -> Shopping cart and checkout process
- style.css              -> Main stylesheet
- img/                   -> Image assets
- script.js              -> JavaScript plugins and main scripts

INSTRUCTIONS:
- Open index.html in any modern browser to preview the homepage.
- Customize the CSS from "style.css".
- You may replace the product images and text with your own.

REQUIREMENTS:
- Basic HTML/CSS knowledge for customization.
- For contact form to work, you'll need to integrate a backend (like PHP or a service like Formspree).

SUPPORT:
- For customization or upgrades (WordPress, Laravel, etc.), please contact your developer or reach out on Fiverr/Upwork.

Thank you!

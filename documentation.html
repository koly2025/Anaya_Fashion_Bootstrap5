Anaya Fashion - Documentation

Thank you for purchasing Anaya Fashion HTML Template. This documentation will guide you to use and customize the template.

Installation

  1. Extract the ZIP file you downloaded.
  2. Open index.html in your browser to preview the website.
  3. Upload all files to your hosting server to make it live.

Folder Structure

Anaya Fashion/
├── css/
├── js/
├── images/
├── index.html
├── shop.html
├── product.html
├── contact.html
└── ...

Customization

You can edit HTML files using any code editor (like VS Code). CSS files are located in the css/ folder and JavaScript in js/.

Ajax Contact Form

The contact form uses Formspree. Update the form action URL in contact.html with your own Formspree endpoint.

Dark Mode

Click the toggle button on the top-right to switch between light and dark mode. JavaScript handles the class toggling.

Product Filtering

Go to shop.html to see the category-based filtering system. Products show/hide based on selected category.
